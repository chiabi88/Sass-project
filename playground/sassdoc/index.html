    get-hypotenuse

    @function get-hypotenuse($ab, $ac) { ... }

    Description

    삼각형의 빗변을 구하는 함수
    원본은 따옴표가 제거된 문자열로 값을 반환해 다음 계산을 할 수 없어서 set-unit 함수 추가해 숫자로 값을 반환하도록 변경함

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $ab

    삼각형 leg

    		Number (with unit)— none
    $ac

    삼각형 leg

    		Number (with unit)— none

    Example

    @debug get-hypotenuse(5px, 5px) // DEBUG: 7.07107px
@debug get-hypotenuse(5px, 5rem) // Error ... 'The two value into the get-hypotenuse() function must share the same unit type'

    Throws

    • The two value into the get-hypotenuse() function must share the same unit type

    Requires

    • [function] get-unit
    • [function] get-unit

    • [function] sqrt
    • [function] square

    • [function] set-unit
