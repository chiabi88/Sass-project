    font

    @mixin font($font-name, $font-path, $font-option) { ... }

    Description

    font-face 믹스인 (eot, woff, ttf, svg)

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $font-name

    폰트 파일 명

    		String— none
    $font-path

    폰트 파일 경로, $font-path, $font-name 조합의 경로를 사용하거나 직접 지정

    		String— none
    $font-option

    font-weight, font-style 값으로 들어갈 리스트, 기본값: normal normal

    		List— none

    Example

    scss

    @include font('NanumBarunGothic');

    css output

    @font-face {
  font-family: "NanumBarunGothic";
  src: url("../font/NanumBarunGothic.eot");
  src: url("../font/NanumBarunGothic.eot?#iefix") format("embaded-opentype"),
       url("../font/NanumBarunGothic.woff") format("woff"), 
       url("../font/NanumBarunGothic.ttf") format("trutype"), 
       url("../font/NanumBarunGothic.svg#NanumBarunGothic") format("svg");
  font-weight: normal;
  font-style: normal;
}
