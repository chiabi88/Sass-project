    thumbnail-grid-list

    @mixin thumbnail-grid-list($thumb-grid-list: 1, $thumb-grid-container: auto, $thumb-grid-columns: 12, $thumb-grid-gutters: 0.25, $thumb-grid-bottom-gutters: 0, $thumb-grid-effect-target: null, $thumb-grid-effect-name: 'zoom-in', $thumb-grid-query-fallback: false, $thumb-grid-output: 'float') { ... }

    Description

    썸네일 리스트 스니펫 믹스인 susy에 의존함

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $thumb-grid-list

    리스트 한줄에 들어갈 아이템 개수

    		Number (unitless)1
    $thumb-grid-container

    susy layout container 값

    		Auto or Number (with unit)auto
    $thumb-grid-columns

    susy layout columns 값

    		Number (unitless)12
    $thumb-grid-gutters

    susy layout gutter 값

    		Number (unitless)0.25
    $thumb-grid-bottom-gutters

    리스트 아이템 margin-bottom 값

    		Number (with unit)0
    $thumb-grid-effect-target

    썸네일 오버 이펙트 필요할 경우 대상이 되는 img의 부모요소 선택자

    		Stringnull
    $thumb-grid-effect-name

    썸네일 오버 이펙트 이름 ['zoom-in' | 'blur']

    		String'zoom-in'
    $thumb-grid-query-fallback

    미디어 쿼리 미지원 폴백 구문 노출여부, 'only-desktop' 믹스인 안에서 사용시 true로 설정

    		Boolfalse
    $thumb-grid-output

    susy layout output 값

    		String'float'

    Example

    scss

    .thumb_3_list{
  /* 썸네일 3개인 리스트: 데스크탑 */
  // @include thumbnail-grid-list(3, auto, 3, 0.16, 46px, '.thumb_photo', 'blur', true);
  @include thumbnail-grid-list(
      $thumb-grid-list: 3,
      $thumb-grid-columns: 3, 
      $thumb-grid-gutters: 0.16, 
      $thumb-grid-bottom-gutters: 46px, 
      $thumb-grid-effect-target: '.thumb_photo',
      $thumb-grid-effect-name: 'blur',
      $thumb-grid-query-fallback: true
  );
}

    css output
